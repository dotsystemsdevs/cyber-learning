# Confidentiality

![CIA Definitions](cia-definitions.png)

> The information is safe from accidental or international disclosure.

---

## What it means
Confidentiality means that only authorized people can see the information. If someone who shouldn't have access reads, copies or steals data — confidentiality is broken.

## Real examples
- A hacker steals a password database
- An employee reads files they are not authorized to see
- Network traffic is intercepted (sniffing)

## Attacks that break Confidentiality
- **Phishing** — trick users into giving up credentials
- **Sniffing** — intercept unencrypted network traffic
- **Credential dumping** — extract passwords from memory

## How it is protected
- Encryption (data unreadable without a key)
- Access control (only authorized users can log in)
- Multi-factor authentication (MFA)
